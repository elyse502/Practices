# JavaScript
## What is JavaScript?
JavaScript is one of the most popular and widely used programming languages in the world right now. It's a great opportunity to get a good job out of learning JavaScript. You can work as a front-end developer or a full stack developer who knows both the front end and the back end.

## What can you do with JavaScript?
For a long time, javascript was only used in browsers to build interactive web pages some developers refer to javascript as a toy language but those days are gone because of huge community support and investments by large companies like Facebook and Google these days you can build full-blown web or mobile apps as well as real time networking applications like chats and video
